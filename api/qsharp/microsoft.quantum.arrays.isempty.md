Retornará true se e somente se uma matriz estiver vazia.

```qsharp
function IsEmpty<'T> (array : 'T[]) : Bool
```


## <a name="input"></a>Entrada

## <a name="output--bool"></a>Saída: [bool](xref:microsoft.quantum.lang-ref.bool)
