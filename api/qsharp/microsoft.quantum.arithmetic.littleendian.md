# <a name="littleendian-user-defined-type"></a>Tipo definido pelo usuário LittleEndian

Namespace: [Microsoft. Quantum. aritmético](xref:Microsoft.Quantum.Arithmetic)

Agrupa [](https://nuget.org/packages/)


Registre-se que codifica um inteiro não assinado na ordem little-endian. O qubit com index `0` codifica o menor bit de um inteiro sem sinal.

```qsharp

newtype LittleEndian = (Qubit[]);
```



## <a name="remarks"></a>Comentários

Abreviamos `LittleEndian` como `LE` na documentação.
