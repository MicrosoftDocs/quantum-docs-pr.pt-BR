# <a name="preparefxp-operation"></a>Operação PrepareFxP

Namespace: [Microsoft. Quantum. aritmético](xref:Microsoft.Quantum.Arithmetic)

Agrupa [](https://nuget.org/packages/)


Inicializar um número de ponto fixo Quantum para uma constante clássica.

```qsharp
operation PrepareFxP (constant : Double, fp : Microsoft.Quantum.Arithmetic.FixedPoint) : Unit
```


## <a name="input"></a>Entrada

### <a name="constant--double"></a>constante: [Double](xref:microsoft.quantum.lang-ref.double)

Constante na qual inicializar o número de ponto fixo Quantum.


### <a name="fp--fixedpoint"></a>fp: [FixedPoint](xref:Microsoft.Quantum.Arithmetic.FixedPoint)

Número de ponto fixo (do tipo FixedPoint) a ser inicializado.
