# <a name="q-libraries"></a>Bibliotecas do Q# #

A biblioteca padrão Q# inclui operações, funções e tipos definidos pelo usuário usados para desenvolver programas quânticos. As três bibliotecas desenvolvidas com base na biblioteca padrão também estão incluídas – as bibliotecas de química, numéricos e aprendizado de máquina.

Para saber mais sobre as bibliotecas Q#, confira <xref:microsoft.quantum.libraries>.
