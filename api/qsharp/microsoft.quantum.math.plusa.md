# <a name="plusa-function"></a>Função PlusA

Namespace: [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Agrupa [](https://nuget.org/packages/)


Retorna a soma (concatenação) de duas entradas.

```qsharp
function PlusA<'Element> (a : 'Element[], b : 'Element[]) : 'Element[]
```


## <a name="input"></a>Entrada

### <a name="a--element"></a>a: ' element []

A primeira entrada $a $ a ser somada.


### <a name="b--element"></a>b: ' element []

A segunda entrada $b $ a ser somada.



## <a name="output--element"></a>Saída: ' element []

A soma $a + b $.

## <a name="type-parameters"></a>Parâmetros de tipo

### <a name="element"></a>' Element

O tipo de cada elemento em cada uma das duas matrizes de entrada.
