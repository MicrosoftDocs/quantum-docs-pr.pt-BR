# <a name="addconstantfxp-operation"></a>Operação AddConstantFxP

Namespace: [Microsoft. Quantum. aritmético](xref:Microsoft.Quantum.Arithmetic)

Agrupa [](https://nuget.org/packages/)


Adiciona uma constante clássica a um número de ponto fixo Quantum.

```qsharp
operation AddConstantFxP (constant : Double, fp : Microsoft.Quantum.Arithmetic.FixedPoint) : Unit
```


## <a name="input"></a>Entrada

### <a name="constant--double"></a>constante: [Double](xref:microsoft.quantum.lang-ref.double)

Constante para adicionar ao número de ponto fixo Quantum.


### <a name="fp--fixedpoint"></a>fp: [FixedPoint](xref:Microsoft.Quantum.Arithmetic.FixedPoint)

Número de ponto fixo ao qual a constante será adicionada.



## <a name="output--unit"></a>Saída: [unidade](xref:microsoft.quantum.lang-ref.unit)
