# <a name="iq-magic-commands"></a>Comandos magic do IQ#

| Comandos magic | Resumo |
|---------------|---------|
| [`%check_kata`](xref:microsoft.quantum.iqsharp.magic-ref.check_kata) | Verifica a implementação de referência para um só teste de kata. |
| [`%chemistry.broombridge`](xref:microsoft.quantum.iqsharp.magic-ref.chemistry.broombridge) | Carrega e retorna a representação de problema de estrutura eletrônica Broombridge de um determinado arquivo .yaml. |
| [`%chemistry.encode`](xref:microsoft.quantum.iqsharp.magic-ref.chemistry.encode) | Codifica um hamiltoniano fermiônico em um formato consumível pelo Q#. |
| [`%chemistry.fh.add_terms`](xref:microsoft.quantum.iqsharp.magic-ref.chemistry.fh.add_terms) | Adiciona termos a um hamiltoniano fermiônico. |
| [`%chemistry.fh.load`](xref:microsoft.quantum.iqsharp.magic-ref.chemistry.fh.load) | Carrega o hamiltoniano fermiônico para um problema de estrutura eletrônica. O problema é carregado de um arquivo ou passado como um argumento. |
| [`%chemistry.inputstate.load`](xref:microsoft.quantum.iqsharp.magic-ref.chemistry.inputstate.load) | Carrega o problema de estrutura eletrônica Broombridge e retorna o estado de entrada selecionado. |
| [`%config`](xref:microsoft.quantum.iqsharp.magic-ref.config) | Permite configurar ou consultar opções de configuração. |
| [`%estimate`](xref:microsoft.quantum.iqsharp.magic-ref.estimate) | Executa uma determinada função ou operação no computador de destino do ResourcesEstimator. |
| [`%kata`](xref:microsoft.quantum.iqsharp.magic-ref.kata) | Executa um só teste. |
| [`%lsmagic`](xref:microsoft.quantum.iqsharp.magic-ref.lsmagic) | Retorna uma lista de todos os comandos magic disponíveis no momento. |
| [`%package`](xref:microsoft.quantum.iqsharp.magic-ref.package) | Fornece a capacidade de carregar um pacote NuGet. O pacote deve estar disponível na lista de fontes do NuGet; normalmente isso inclui nuget.org |
| [`%performance`](xref:microsoft.quantum.iqsharp.magic-ref.performance) | Relata as métricas de desempenho atuais desse kernel. |
| [`%simulate`](xref:microsoft.quantum.iqsharp.magic-ref.simulate) | Executa uma determinada função ou operação no computador de destino do QuantumSimulator |
| [`%toffoli`](xref:microsoft.quantum.iqsharp.magic-ref.toffoli) | Executa uma determinada função ou operação no computador de destino do simulador ToffoliSimulator |
| [`%who`](xref:microsoft.quantum.iqsharp.magic-ref.who) | Fornece ações relacionadas ao workspace atual. |
| [`%workspace`](xref:microsoft.quantum.iqsharp.magic-ref.workspace) | Retorna uma lista de todas as operações e funções definidas na sessão atual, interativamente ou carregadas do workspace atual. |
