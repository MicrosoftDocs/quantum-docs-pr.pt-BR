# <a name="dirac-notation"></a><span data-ttu-id="86d6b-182">Notação de Dirac</span><span class="sxs-lookup"><span data-stu-id="86d6b-182">Dirac Notation</span></span>

<span data-ttu-id="86d6b-183">Embora a notação de vetor de coluna seja onipresente em Algebra linear, ela geralmente é complicada na computação Quantum, especialmente ao lidar com vários qubits.</span><span class="sxs-lookup"><span data-stu-id="86d6b-183">While column vector notation is ubiquitous in linear algebra, it is often cumbersome in quantum computing especially when dealing with multiple qubits.</span></span>  <span data-ttu-id="86d6b-184">Por exemplo, quando definimos $ \psi $ para ser um vetor, ele não está explicitamente claro se $ \psi $ é um vetor de linha ou de coluna.</span><span class="sxs-lookup"><span data-stu-id="86d6b-184">For example, when we define $\psi$ to be a vector it is not explicitly clear whether $\psi$ is a row or a column vector.</span></span>  <span data-ttu-id="86d6b-185">Portanto $ \phi $ , se e $ \psi $ forem vetores, ele será igualmente desmarcado se $ \phi \psi $ for par definido, pois as formas de $ \phi $ e $ \psi $ podem não estar claras no contexto.</span><span class="sxs-lookup"><span data-stu-id="86d6b-185">Thus if $\phi$ and $\psi$ are vectors then it is equally unclear if $\phi \psi$ is even defined because the shapes of $\phi$ and $\psi$ may be unclear in the context.</span></span>  <span data-ttu-id="86d6b-186">Além da ambiguidade sobre as formas de vetores, expressar até mesmo vetores simples usando a notação algébricas linear introduzida anteriormente pode ser muito complicado.</span><span class="sxs-lookup"><span data-stu-id="86d6b-186">Beyond the ambiguity about the shapes of vectors, expressing even simple vectors using the linear algebraic notation introduced earlier can be very cumbersome.</span></span> <span data-ttu-id="86d6b-187">Por exemplo, se quisermos descrever um $ estado n $ -qubit, em que cada qubit usa o valor $ 0 $ , formalmente expressos o estado como</span><span class="sxs-lookup"><span data-stu-id="86d6b-187">For example, if we wish to describe an $n$-qubit state where each qubit takes the value $0$ then we would formally express the state as</span></span> 

<span data-ttu-id="86d6b-188">$$\begin{bmatrix}1 \\\\ 0 \end{bmatrix} \otimes \cdots \otimes \begin{bmatrix} 1 \\\\ 0 \end{bmatrix} .</span><span class="sxs-lookup"><span data-stu-id="86d6b-188">$$\begin{bmatrix}1 \\\\  0 \end{bmatrix}\otimes \cdots \otimes\begin{bmatrix}1 \\\\  0 \end{bmatrix}.</span></span> $$  

<span data-ttu-id="86d6b-189">É claro que avaliar esse produto tensor é impraticável porque o vetor está em um espaço exponencialmente grande e, portanto, essa notação é, na verdade, a melhor descrição do estado que pode ser fornecido usando a notação anterior.</span><span class="sxs-lookup"><span data-stu-id="86d6b-189">Of course evaluating this tensor product is impractical because the vector lies in an exponentially large space and so this notation is in fact the best description of the state that can be given using the previous notation.</span></span>  

<span data-ttu-id="86d6b-190">A [*notação Dirac*](https://en.wikipedia.org/wiki/Bra%E2%80%93ket_notation) resolve esses problemas apresentando uma nova linguagem para se adequar às necessidades exatas da mecânica quantum.</span><span class="sxs-lookup"><span data-stu-id="86d6b-190">[*Dirac notation*](https://en.wikipedia.org/wiki/Bra%E2%80%93ket_notation) solves these issues by presenting a new language to fit the precise needs of quantum mechanics.</span></span>  <span data-ttu-id="86d6b-191">Por esse motivo, recomendamos que o leitor não exiba os exemplos nesta seção como uma receita rígida de como descrever os Estados da Quantum, mas, em vez disso, incentive o leitor a exibi-los como sugestões que podem ser usadas para expressar as ideias da Quantum de forma concisa.</span><span class="sxs-lookup"><span data-stu-id="86d6b-191">For this reason, we recommend the reader not view the examples in this section as a rigid prescription of how to describe quantum states, but rather encourage the reader to view these as suggestions that can be used to concisely express quantum ideas.</span></span>

<span data-ttu-id="86d6b-192">Há dois tipos de vetores na notação Dirac: o vetor *Bra* e o vetor *ket* , portanto, nomeados porque, quando agrupados, formam um produto de *freio* ou interno.</span><span class="sxs-lookup"><span data-stu-id="86d6b-192">There are two types of vectors in Dirac notation: the *bra* vector and the *ket* vector, so named because when put together they form a *braket* or inner product.</span></span>  <span data-ttu-id="86d6b-193">Se $ \psi $ for um vetor de coluna, poderá escrevê-lo na notação Dirac como $ \ket { \psi } $ , em que o $ \ket { \cdot denota que se trata de } $ um vetor de coluna de unidade, ou seja, um vetor *ket* .</span><span class="sxs-lookup"><span data-stu-id="86d6b-193">If $\psi$ is a column vector then we can write it in Dirac notation as $\ket{\psi}$, where the $\ket{\cdot}$ denotes that it is a unit column vector, i.e., a *ket* vector.</span></span>  <span data-ttu-id="86d6b-194">Da mesma forma, o vetor de linha $ \psi ^ \dagger $ é expresso como $ \bra { \psi } $ .</span><span class="sxs-lookup"><span data-stu-id="86d6b-194">Similarly, the row vector $\psi^\dagger$ is expressed as $\bra{\psi}$.</span></span> <span data-ttu-id="86d6b-195">Em outras palavras, $ \psi ^ \dagger $ é obtido pela aplicação de um conjugado complexo de entrada aos elementos da transpoção de $ \psi $ .</span><span class="sxs-lookup"><span data-stu-id="86d6b-195">In other words, $\psi^\dagger$ is obtained by applying entry-wise complex conjugation to the elements of the transpose of $\psi$.</span></span> <span data-ttu-id="86d6b-196">A notação bra-ket implica diretamente que $ \braket { \psi | \psi } $ é o produto interno do vetor $ \psi $ com ele mesmo, que é por definição $ 1 $ .</span><span class="sxs-lookup"><span data-stu-id="86d6b-196">The bra-ket notation directly implies that $\braket{\psi|\psi}$ is the inner product of vector $\psi$ with itself, which is by definition $1$.</span></span>  

<span data-ttu-id="86d6b-197">Em geral, se $ \psi $ e $ \phi $ são vetores de estado Quantum, seu produto interno é $ \braket { \phi | \psi } $ o que implica que a probabilidade de medir o estado $ \ket { \psi } $ como $ \ket { \phi } $ é $ | \braket { \phi | \psi } | ^ 2 $ .</span><span class="sxs-lookup"><span data-stu-id="86d6b-197">More generally, if $\psi$ and $\phi$ are quantum state vectors their inner product is $\braket{\phi|\psi}$ which implies that the probability of measuring the state $\ket{\psi}$ to be $\ket{\phi}$ is $|\braket{\phi|\psi}|^2$.</span></span>  

<span data-ttu-id="86d6b-198">A Convenção a seguir é usada para descrever os Estados do Quantum que codificam os valores de zero e um (os Estados de base computacional de qubit único):</span><span class="sxs-lookup"><span data-stu-id="86d6b-198">The following convention is used to describe the quantum states that encode the values of zero and one (the single-qubit computational basis states):</span></span>

$$
<span data-ttu-id="86d6b-199">\begin{bmatrix}1 \\\\ 0 \end{bmatrix} = \ket { 0 } ,\qquad</span><span class="sxs-lookup"><span data-stu-id="86d6b-199">\begin{bmatrix} 1 \\\\  0 \end{bmatrix} = \ket{0},\qquad</span></span>
<span data-ttu-id="86d6b-200">\begin{bmatrix}0 \\\\ 1 \end{bmatrix} = \ket { 1 } .</span><span class="sxs-lookup"><span data-stu-id="86d6b-200">\begin{bmatrix} 0 \\\\  1 \end{bmatrix} = \ket{1}.</span></span>
$$

### <a name="example-representing-the-hadamard-operation-with-dirac-notation"></a><span data-ttu-id="86d6b-201">Exemplo: representando a operação Hadamard com notação Dirac</span><span class="sxs-lookup"><span data-stu-id="86d6b-201">Example: representing the Hadamard operation with Dirac notation</span></span>

<span data-ttu-id="86d6b-202">A notação a seguir é geralmente usada para descrever os Estados resultantes da aplicação da porta Hadamard para $ \ket { 0 } $ e $ \ket { 1 } $ (que correspondem aos vetores de unidade nas $ direções + x $ e $ -x $ na esfera Bloch):</span><span class="sxs-lookup"><span data-stu-id="86d6b-202">The following notation is often used to describe the states that result from applying the Hadamard gate to $\ket{0}$ and $\ket{1}$ (which correspond to the unit vectors in the $+x$ and $-x$ directions on the Bloch sphere):</span></span>

$$
<span data-ttu-id="86d6b-203">\frac{1 } { \sqrt { 2 } } \begin{bmatrix} 1 \\\\ 1 \end{bmatrix} = H \ket { 0 } = \ket { + } ,\qquad</span><span class="sxs-lookup"><span data-stu-id="86d6b-203">\frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\\\  1 \end{bmatrix}=H\ket{0} = \ket{+},\qquad</span></span>
<span data-ttu-id="86d6b-204">\frac{1 } { \sqrt { 2 } } \begin{bmatrix} 1 \\\\ -1 \end{bmatrix} = H \ket { 1 } = \ket { - } .</span><span class="sxs-lookup"><span data-stu-id="86d6b-204">\frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\\\  -1 \end{bmatrix} =H\ket{1} = \ket{-} .</span></span>
$$

<span data-ttu-id="86d6b-205">Esses Estados também podem ser expandidos usando a notação Dirac como somas de $ \ket { 0 } $ e $ \ket { 1 } $ :</span><span class="sxs-lookup"><span data-stu-id="86d6b-205">These states can also be expanded using Dirac notation as sums of $\ket{0}$ and $\ket{1}$:</span></span>

$$
<span data-ttu-id="86d6b-206">\ket{+}= \frac{ 1 } { \sqrt { 2 } } ( \ket { 0 }  +  \ket { 1 } ), \qquad \ket { - } = \frac { 1 } { \sqrt { 2 } } ( \ket { 0 }  -  \ket { 1 } ).</span><span class="sxs-lookup"><span data-stu-id="86d6b-206">\ket{+} = \frac{1}{\sqrt{2}}(\ket{0} + \ket{1}),\qquad \ket{-} = \frac{1}{\sqrt{2}}(\ket{0} - \ket{1}).</span></span>
