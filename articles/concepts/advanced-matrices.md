$$
<span data-ttu-id="856f3-210">e ^ A = \boldone + a + \frac { a ^ 2 } { 2! } + \frac { Um ^ 3 3 } { !}+\cdots</span><span class="sxs-lookup"><span data-stu-id="856f3-210">e^A=\boldone + A + \frac{A^2}{2!}+\frac{A^3}{3!}+\cdots</span></span>
