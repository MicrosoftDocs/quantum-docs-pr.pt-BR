<span data-ttu-id="6069f-139">O código a seguir ilustra alguns dos novos recursos:</span><span class="sxs-lookup"><span data-stu-id="6069f-139">The following code illustrates some of the new features:</span></span>

        // Mutable assignments can now destructure and ignore parts of tuples,
