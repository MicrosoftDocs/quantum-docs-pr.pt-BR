# <a name="what-is-q"></a><span data-ttu-id="e5084-103">O que é o Q#?</span><span class="sxs-lookup"><span data-stu-id="e5084-103">What is Q#?</span></span>

<span data-ttu-id="e5084-104">O Q# é uma linguagem de programação com recursos que são especiais à computação quântica.</span><span class="sxs-lookup"><span data-stu-id="e5084-104">Q# is a programming language with features that are special to quantum computing.</span></span> <span data-ttu-id="e5084-105">O Q# fornece aos programadores quânticos uma estrutura que permite que você se concentre nos algoritmos sem precisar se preocupar com detalhes técnicos, como a otimização da sequência de portão ou a implementação física de um computador quântico.</span><span class="sxs-lookup"><span data-stu-id="e5084-105">Q# provides quantum programmers a framework that allows you to focus on the algorithms without having to care about technical details like gate sequence optimization or the physical implementation of a quantum computer.</span></span>

<span data-ttu-id="e5084-106">A linguagem de programação Q# fornece um conjunto intuitivo de tipos, operações e expressões lógicas para desenvolver algoritmos sem precisar se preocupar com a lógica interna do computador quântico.</span><span class="sxs-lookup"><span data-stu-id="e5084-106">The Q# programming language provides you with an intuitive set of types, operations and logic expressions to develop algorithms without having to worry about the internal logic of the quantum computer.</span></span>

## <a name="code-algorithms"></a><span data-ttu-id="e5084-107">Algoritmos de código</span><span class="sxs-lookup"><span data-stu-id="e5084-107">Code algorithms</span></span>

<span data-ttu-id="e5084-108">Nos primórdios da computação quântica, os algoritmos eram visualizados como diagramas da mesma forma que os diagramas de circuito na computação clássica.</span><span class="sxs-lookup"><span data-stu-id="e5084-108">In the early days of quantum computing algorithms were visualized as diagrams similarly to circuit diagrams in classical computing.</span></span>  <span data-ttu-id="e5084-109">Embora o modelo de circuito tenha sido muito útil por muitos anos na pesquisa da computação quântica, aqui na Microsoft, acreditamos que os desenvolvedores podem ir além dos circuitos quânticos e desenvolver aplicativos e algoritmos quânticos usando o Q#.</span><span class="sxs-lookup"><span data-stu-id="e5084-109">While the circuit model has been very useful for many years in quantum computing research, here at Microsoft, we believe that developers can go beyond quantum circuits and develop quantum algorithms and applications using Q#.</span></span> <span data-ttu-id="e5084-110">A linguagem Q# foi criada para aproveitar o que aprendemos ao longo de décadas de desenvolvimento de software clássico e capacitar os desenvolvedores quânticos com uma funcionalidade de linguagem de alto nível especificamente destinada à computação quântica.</span><span class="sxs-lookup"><span data-stu-id="e5084-110">The Q# language was built to take advantage of what we’ve learned through decades of classical software development, and empower quantum developers with high-level language functionality specifically targeted for quantum computing.</span></span>


## <a name="how-does-q-work"></a><span data-ttu-id="e5084-111">Como funciona o Q#?</span><span class="sxs-lookup"><span data-stu-id="e5084-111">How does Q# work?</span></span>

<span data-ttu-id="e5084-112">Um dos blocos de construção fundamentais do Q# é o tipo `Qubit`, que não pode ser copiado nem acessado diretamente, assim como um qubit real.</span><span class="sxs-lookup"><span data-stu-id="e5084-112">One of the fundamental building blocks of Q# is the `Qubit` type, which cannot be copied or directly accessed, just like a real qubit.</span></span> <span data-ttu-id="e5084-113">Em vez disso, podemos medi-lo e armazenar o resultado da medida em uma variável `Result`, um tipo Q# que pode usar dois valores possíveis: `Zero` e `One`.</span><span class="sxs-lookup"><span data-stu-id="e5084-113">Instead, we can measure it and store the outcome of the measurement in a `Result` variable, a Q# type that can take two possible values: `Zero` and `One`.</span></span> <span data-ttu-id="e5084-114">Constructos como esse garantem que os algoritmos sempre respeitem as leis da física quântica e possam ser executados corretamente em simuladores ou computadores quânticos.</span><span class="sxs-lookup"><span data-stu-id="e5084-114">Constructs like this one guarantee that algorithms always respect the laws of quantum physics and can run correctly on quantum computers or simulators.</span></span>

<span data-ttu-id="e5084-115">O Q# também inclui recursos lógicos clássicos como condicionais ou loops com algumas sutilezas para garantir que todas as regras quânticas sejam respeitadas.</span><span class="sxs-lookup"><span data-stu-id="e5084-115">Q# also includes classical logic features like conditionals or loops with some subtleties to make sure that all the quantum rules are being respected.</span></span> <span data-ttu-id="e5084-116">Por exemplo, as operações quânticas precisam ser reversíveis.</span><span class="sxs-lookup"><span data-stu-id="e5084-116">For example, quantum operations need to be reversible.</span></span> <span data-ttu-id="e5084-117">Isso impõe algumas restrições sobre o modo como os loops são executados.</span><span class="sxs-lookup"><span data-stu-id="e5084-117">This enforces some constraints on the way loops are executed.</span></span>

<span data-ttu-id="e5084-118">Os programas Q# geralmente são emparelhados com um programa host escrito em C# ou no Python, que possa fornecer uma organização conveniente dos códigos clássico e quântico.</span><span class="sxs-lookup"><span data-stu-id="e5084-118">Q# programs are often paired with a host program written in C# or Python, which can provide convenient organization of classical and quantum code.</span></span> <span data-ttu-id="e5084-119">Além de dar suporte a linguagens .NET, como C# e Python, o QDK fornece suporte do Jupyter Notebook com o kernel IQ# Jupyter.</span><span class="sxs-lookup"><span data-stu-id="e5084-119">In addition to supporting .NET languages such as C# and Python, the QDK provides Jupyter Notebook support with the IQ# Jupyter kernel.</span></span>

## <a name="use-q-to-design-quantum-algorithms"></a><span data-ttu-id="e5084-123">Usar o Q# para criar algoritmos quânticos</span><span class="sxs-lookup"><span data-stu-id="e5084-123">Use Q# to design quantum algorithms</span></span>

<span data-ttu-id="e5084-124">O Q# fornece um número cada vez maior de bibliotecas e tipos definidos pelo usuário que ajudarão você a implementar ferramentas e a criar algoritmos quânticos avançados.</span><span class="sxs-lookup"><span data-stu-id="e5084-124">Q# provides you with an increasing number of libraries and user-defined types that will help you to implement tools and build advanced quantum algorithms.</span></span> <span data-ttu-id="e5084-125">Por exemplo, você precisa entrelaçar os dois qubits q1 e q2?</span><span class="sxs-lookup"><span data-stu-id="e5084-125">For example, you need to entangle two-qubits q1 and q2?</span></span> <span data-ttu-id="e5084-126">Em vez de aplicar individualmente os portões necessários para entrelaçar os qubits, você pode usar a operação já interna `PrepareEntangledState([q1], [q2])`.</span><span class="sxs-lookup"><span data-stu-id="e5084-126">Instead of applying individually the necessary gates to get the qubits entangled you can use the already built-in operation `PrepareEntangledState([q1], [q2])`.</span></span>
